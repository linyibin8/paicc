# PAI-CC V3.0 学习伙伴 AI 伴侣 - 开发工作流

## 项目概述

**PAI-CC** 是一个面向学生学习陪伴的拍题/智能连拍辅助系统，结合语音识别、TTS 语音合成和手势识别。

### 核心功能
- 📷 **iPhone 摄像端**: 持续观察课本、试卷或屏幕学习场景
- 🖐️ **手势识别**: 伸出食指指向题目开始问答，OK 手势打断，✌️ 结束
- 🎤 **语音交互**: 语音问答 + TTS 回复
- 🤖 **AI 问答**: 基于当前画面 + 语音输入，生成答案
- 📊 **学习报告**: 生成回合报告、错题候选、知识点清单

## 技术架构

### 后端 (FastAPI)
- **API 服务**: http://100.64.0.13:8090
- **AI 模型**: http://100.64.0.5:39000/v1 (evowit-agent27b)
- **TTS 服务**: http://127.0.0.1:8880 (Kokoro)

### iOS 端
- **语言**: Swift
- **框架**: UIKit + Vision + AVFoundation
- **功能模块**:
  - CameraService: 相机视频流采集
  - GestureService: Vision 手势识别
  - VoiceService: 语音识别 + TTS
  - QAService: 问答状态机
  - QAOverlayView: UI 浮层

## AI 问答流程

```
扫描中 → 伸出食指指向题目
       ↓
检测到指向手势（连续4帧）→ 截取当前帧
       ↓
TTS 说 "请说" → 麦克风打开
       ↓
用户说 "这道题怎么做？" → 3秒沉默自动结束
       ↓
TTS 说 "好的" → 思考音开始播放
       ↓
图片 + 上下文 + 语音文本 → 后端 AI → 大模型分析
       ↓
收到回答 → 停止思考音 → TTS 朗读答案 → 15秒后浮层自动消失
       ↓
用户可随时 👌 OK 打断并追问 → 或 ✌️ 结束本轮
```
